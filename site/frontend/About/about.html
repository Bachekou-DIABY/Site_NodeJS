Accueil Carte Mix Energétique Statistiques Consommation Electrique A Propos

A PROPOS Principe du projet : Le principe de ce projet est de Choisir une source de données accessible (open data, data crawling, etc), de collecter ces données, et de les analyser. Nous avons choisi d'étudier les différentes sources d'énergies dans le monde ainsi que leur production a l'echelle française. Nous avons aussi mis en place un outil permettant de consulter sa consommation électrique si l'on possède un compteur Linky. Divers graphes sont présents sur la page statistiques offrent quelques informations générales sur les différentes sources d'énergies dans le monde.
